## 如何使用

*强烈建议阅读：[Jekyll 中文文档](http://jekyllcn.com/docs/home/)*

第一步： 点击右上角 fork，即拷贝该主题代码到你的账户。

第二步： 更改你的仓库中该项目的名字 haiyung.github.io 为 your_username.github.io。

接下来，你就可以按照 Jekyll 的标准书写自己的文章啦～～～
